## PHASE 5 — EMBASSY & VISA PROCESS (Country-Specific)

### UAE Work Visa Process (Most Common):
1. Employer applies for Labour Approval (MOL/MOHRE)
2. Entry Permit issued (valid 60 days — fly before expiry)
3. Medical fitness test in UAE (approved HAAD/DHA centre)
4. Emirates ID biometrics
5. Visa stamping in passport / residence permit
6. MOHRE contract registration
**Timeline:** 4–8 weeks from job offer to visa stamp

### Saudi Arabia Work Visa (Iqama):
1. Employer applies for work visa block from MOL
2. MOFA Saudi approves — visa number issued
3. Medical test in Pakistan (approved centre)
4. Biometrics at Saudi Embassy Islamabad/Karachi
5. Employment visa stamped in passport
6. Fly to KSA → Iqama issued within 90 days
**Timeline:** 6–12 weeks

### Qatar Work Visa (QID):
1. Employer files application with Ministry of Interior
2. Entry permit via Metrash2 / MDPS portal
3. Medical test in Qatar
4. Qatar ID (QID) issued
**Timeline:** 4–10 weeks
